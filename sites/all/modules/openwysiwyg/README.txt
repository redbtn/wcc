What is openWYSIWYG?
----------------
- Free cross-browser WYSIWYG Editor!
- Very fast WYSIWYG editor
- Built in File Browser! Browse and create sub-folder
- Built in Image operation: Resize image
- Support Toolbar Buttons set: full or custom toolbar

Installation:
-------------
Installation Instructions:

1. Extract to sites/all/module/openwysiwyg/
2. Enable the openWYSIWYG module in Drupal (admin/build/modules).
   

Configuration:
--------------
Go to Administer - Site Configuration - OpenWYSIWYG

Tips:
-----
If you need to separate teaser and body then please install excerpt module (http://drupal.org/project/excerpt/)
